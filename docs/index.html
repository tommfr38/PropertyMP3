Everything you need to tag your music

Open a file, fix the details, save. That's it.

Drag & drop

Drop an MP3 straight onto the window, or pick one with the file dialog.

Edit every tag

Title, artist, album, album artist, year, genre, track #, disc # and comments.

Album art

Click or drop a JPG/PNG to set the embedded cover image, with a smooth cross-fade.

File details at a glance

See bitrate, sample rate, channels, duration and file size without leaving the editor.

Spotify-ready output

Saves as ID3v2.4 — the tag format Spotify reads most reliably for local files.

Smooth & native

60 fps animations and a green focus glow, built in C++ with Qt 6 for a fast, native feel.
